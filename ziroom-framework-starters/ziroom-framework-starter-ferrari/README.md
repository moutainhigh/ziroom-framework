#Ferrari
MQ工具对发送和消费MQ消息进行落地存储

## 主要功能

    支持producer和consumer消息持久化
    支持producer发送失败重试

### RabbitMQ
> RabbitMQ: 只支持org.springframework.amqp 方式的rabbitMq接入
